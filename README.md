🎯 Что это за проект

Это персональная медиа-платформа (portfolio / media CMS)

Сайт + система управления контентом:
- видео
- фото
- (в будущем: блог, музыка и т.д.)

👉 Важно:

Это не просто сайт, а архитектура, которую можно масштабировать
🧠 Главная идея проекта
Разделы → Альбомы → Медиа → Просмотр

Ты строишь универсальную систему:

добавил новый раздел → всё работает
добавил новый тип контента → не переписываешь проект
🧱 Архитектура проекта

Проект состоит из 4 слоёв:

1. Frontend (Next.js)

👉 интерфейс пользователя

2. Backend (Next API)

👉 логика:

upload
создание медиа
получение данных
3. Storage (S3 / Yandex Cloud)

👉 хранение файлов

media/
  videos/
  photos/
4. Database (PostgreSQL + Prisma)

👉 структура данных:

Section → Album → MediaFile → Video
