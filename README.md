1. Главная страница

Это имиджевая fullscreen-страница, задача которой — сразу впечатлить.

Что на ней должно быть:

fullscreen видео-фон
видео берётся из специального альбома с лучшими роликами
крупный эмоциональный заголовок поверх
слева сверху — навигация
справа сверху — логотип
справа снизу — твоя PNG/картинка с пультом или в очках
ещё один объект, например дрон, появляется с задержкой, будто “влетает” в сцену
курсор на главной не обычный, а в виде руки, чтобы было понятно: экран интерактивный
при клике по hero-видео показывается следующее видео из подборки

То есть это не просто “страница”, а живой entrance screen в проект 🎬

2. Раздел Видео

Это уже не лендинг, а рабочий viewer-каталог.

Структура такая:

сверху та же навигация, что и на главной
слева список альбомов видео
каждый альбом — одинаковый прямоугольный блок
при клике на альбом слева вместо списка альбомов показывается список видео этого альбома
над списком появляется кнопка «Альбомы», чтобы вернуться назад
при наведении на видео-preview оно начинает проигрываться
при клике выбранное видео показывается справа
справа сверху над видео:
название альбома
название выбранного видео
само видео справа большое, с обычными нативными контролами
можно открыть fullscreen
если пользователь только зашёл в альбом и ничего не выбрал — справа показывается случайное видео из альбома

То есть логика такая:

Альбомы → Видео внутри альбома → Просмотр выбранного видео справа

3. Раздел Фото

Почти полностью повторяет видео-раздел.

Разница:

слева альбомы фото
при входе в альбом слева показываются фото-превью
справа — большое выбранное фото
над большим фото — название альбома и название фото
в нижнем левом углу большого изображения — декоративная картинка человека с рюкзаком

То есть это тот же viewer, но уже для photo storytelling 📷

4. Раздел Блог

Пока ты хочешь сделать по той же логике, чтобы не ломать структуру проекта.

То есть временно:

слева список материалов / категорий / статей
справа контент активной статьи

Это разумно, потому что сначала лучше выстроить общий каркас проекта, а потом уже дорабатывать блог отдельно.

6. Админка

Вот тут очень важный момент:
ты хочешь, чтобы админка визуально была почти зеркалом сайта.

И это хорошая идея 💡

Не отдельная “техническая панель”, а такой же аккуратный интерфейс, только с функциями управления.

Главная админки

Пока без точного сценария, но можно сделать dashboard:

количество альбомов
количество видео
количество фото
количество постов
количество музыки

Разделы админки

Например, раздел Фото в админке:

сверху — та же навигация
слева сверху — прямоугольник с формой создания альбома
ниже — список уже созданных альбомов
при выборе альбома справа:
название альбома сверху слева
кнопка загрузки справа сверху
если файлов нет — сообщение “В этом альбоме пока нет медиа”
если файлы есть — показываются плиткой
у каждой плитки:
название файла
кнопка удалить

И ту же модель переносим на видео, блог, музыку.
